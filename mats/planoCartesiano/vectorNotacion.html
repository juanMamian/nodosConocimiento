Los vectores suelen ser identificados mediante una letra. Esa letra puede estar:

  • En negrita
  • Marcada con un pequeña flecha sobre la letra:

Un vector es descrito completamente mediante una magnitud y una dirección.

{{vector.simbolo}}⃗ =({{vector.magnitud}}, {{vector.direccion}}°)
El módulo de un vector se escribe entre barras verticales:
| v⃗ |
| {{vector.simbolo}} ⃗ |={{vector.magnitud}}
El módulo de un vector es siempre un número positivo
