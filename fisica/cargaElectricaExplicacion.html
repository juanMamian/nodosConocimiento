La observación de algunos fenómenos misteriosos llevó a algunas personas a proponer una hipotesis sorprendente.

Todos los cuerpos contienen una sustancia oculta llamada carga eléctrica.

Cuando un cuerpo:

  • Tiene una cantidad normal de carga eléctrica se dice que está neutro.
  • Tiene una cantidad excesiva de carga eléctrica se dice que está cargado positivamente.
  • Tiene un déficit de carga eléctrica se dice que está cargado negativamente.
Al frotar un objeto de vidrio con el pelaje de un animal, se observa que algunos otros objetos experimentan una atracción hacia él. Especialmente si están cerca.
Lo mismo pasa con la goma. Los objetos de goma que son frotados con el pelaje de un animal atraen a algunos otros objetos. Especialmente cuando están cerca.
