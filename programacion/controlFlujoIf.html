La instrucción especial if señala que un bloque de instrucciones debe ejecutarse condicionalmente

La instrucción if incluye una expresiónque se resuelve en un valor Booleano. Las instrucciones de un bloque señalado en un if se ejecutan si su expresión se resuelve en true.

Para escribir una instruccion if se debe:

  • Escribir la palabra clave if.
  • Escribir la expresión.
  • Escribir el bloque de instrucciones que se ejecutarán condicionalmente.
Se quiere escribir un saludo especial para un usuario condicionalmente. Sólo si el nombre del usuario es especial.
if (nombreUsuario == 'Nabucodonosor')
